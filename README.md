# 🛒 Shoplix - Full Stack MERN E-Commerce Platform

Shoplix is a modern full-stack e-commerce web application built using the MERN Stack. It provides a seamless online shopping experience with secure authentication, product management, shopping cart, wishlist, Razorpay payment integration, order management, and an admin dashboard.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/girishposture1623/Ecom-WebApp.git
```

### Go to Project

```bash
cd Ecom-WebApp
```

### Install Dependencies

```bash
npm run install-all
```

### Environment Variables

Create `.env` inside the **backend** folder.

```
MONGO_URI=
JWT_SECRET=
EMAIL_USER=
EMAIL_PASS=
RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
FRONTEND_URL=http://localhost:3000
NODE_ENV=development
```

### Run Development Server

```bash
npm run dev
```
